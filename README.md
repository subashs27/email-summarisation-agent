# Email Summarization Agent

An AI-powered Email Summarization Agent built using Python, Google Gemini API, and Streamlit.

## Features

- Summarizes emails in 2–3 sentences
- Extracts key points
- Identifies action items
- Detects deadlines
- Classifies urgency (Low/Medium/High)
- Interactive Streamlit interface
- Supports TXT, PDF, and DOCX files

## Tech Stack

- Python
- Google Gemini API
- Streamlit
- PyPDF2
- python-docx

## Installation

pip install -r requirements.txt

## Run

streamlit run app.py
